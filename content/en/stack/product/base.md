UI设计常用软件：
+ PS:Adobe PhotoShop
+ AI:Adobe Illustrator
+ Sketch(2010)
+ Adobe XD(2017)

Figma 云服务
特点：
+ 自动布局
+ 官方社区
+ 开发模式

Figma成功后相似软件接连上线：
+ JS Design
+ MasterGo
+ Pixso
+ Motiff

官网：`www.figma.com`

## 5大版块
+ 资源管理 /ResourceManagement

+ 官方社区 /OfficialCommunity

+ 设计模块 /DesignModule

+ 白板模块 /WhiteboardModule

+ 幻灯片模块 /SlideModule

## 4大界面模式
设计模块中包含不同的界面模式

+ 设计模式
左侧：资源面板（页面、图层相关）
右侧：属性面板（元素属性相关）
中间：画布区域
下侧：常用工具

+ 交互模式

+ 开发模式

+ 演示模式


## 工具使用
```text
Shift + 1 (宏观显示所有)
Shift + 0 (100%显示属性)
```
### 选择工具
<font color='cyan'>**Move**</font>:
选择，按住空格移动
```text
V
```
<font color='cyan'>**Handtool**</font>:
移动，画布拖拽
```text
H
```
<font color='cyan'>**Scale**</font>:
放大，等比放大
```text
K
```
### Frame工具
<font color='cyan'>**Frame**</font>:
画板，画框
```text
F
```
<font color='cyan'>**Section**</font>:
分区，一般会反所有画板打包成集合
```text
Shift + S
```
<font color='cyan'>**Slice**</font>:
切片，框一个区域，导出
```text
S
```
### 图形工具
<font color='cyan'>**Rectangle**</font>:
矩形，
```text
R
```

```text
Shift + 图形  各边相等
```
<font color='cyan'>**Line**</font>:
线，
```text
L
```
<font color='cyan'>**Arrow**</font>:
箭头，
```text
Shift + L
```
<font color='cyan'>**Ellipse**</font>:
圆形，
```text
O
```
<font color='cyan'>**Polygon**</font>:
三角形，
<font color='cyan'>**Star**</font>:
星形，
<font color='cyan'>**Image/video**</font>:
图片，
```text
Shift + command + K
```


### 钢笔工具(少用)
<font color='cyan'>**Pen**</font>:

P

<font color='cyan'>**Pencil**</font>:
